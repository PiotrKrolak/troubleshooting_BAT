1. Żeby batch zadziałał prawidłowo musisz mieć zainstalowanego GIT'a
2. Batch należy przenieść na dysk lokalny, dopiero w tedy uruchomić - inaczej repozytoriza nie zostaną pobrane/zaktualizowane
3. Przy spełnieniu powyższych warunków, po uruchomeniu matcha (TROUBLESHOOTING.BAT) pojawi się folder 'troubleshooting' w lokacji C:\\
4. Przed użyciem troubleshootingu, należy uruchomić batch 'TROUBLESHOOTING.BAT' znajdujący się w folderze 'C:\\troubleshooting' w celu zaktualizowania zawartości dokumentów
